Про проект

Інформація про курсову роботу та використані технології

Система управління ресурсами "DevSystem"

Цей проект є курсовою роботою з дисципліни "Веб-технології та веб-програмування". Основна мета системи — надати зручний інтерфейс для адміністрування бази даних користувачів, товарів та замовлень.

Технологічний стек:

  • Backend: Java Spring Boot, Spring Data JPA
  • Database: PostgreSQL
  • Frontend: HTML5, CSS3 (Modern UI/UX), Vanilla JavaScript
  • API: RESTful API, Fetch API

Основні можливості:

Повний цикл CRUD операцій для користувачів.

Каталог товарів з можливістю додавання нових позицій.

Обробка замовлень та прив'язка їх до клієнтів.
