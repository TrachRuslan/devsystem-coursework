Інформація про курсову роботу

Опис системи

Даний проєкт є курсовою роботою на тему "Система управління користувачами та замовленнями". Система розроблена для автоматизації процесів обліку клієнтів та їхніх покупок.

Використані технології:

  • Backend: Java 17, Spring Boot, Spring Data JPA
  • Database: PostgreSQL
  • Frontend: HTML5, CSS3, Vanilla JavaScript, Fetch API
  • Icons: Font Awesome
  • Fonts: Google Inter Font

Основні можливості:

  • Реєстрація та повне керування профілями користувачів.
  • Створення замовлень та прив'язка їх до конкретних клієнтів.
  • Аналітичний дашборд з ключовими показниками системи.
  • Автоматичний розрахунок фінансової статистики.
  • Адаптивний дизайн для зручної роботи на різних пристроях.

Академічна інформація

Тема: Розробка веб-орієнтованої інформаційної системи управління клієнтською базою.

Дисципліна: Об'єктно-орієнтоване програмування / Веб-технології.
